HTML Introduction

Conceito

HTML significa HyperText Markup Language, ou linguagem de marcação de hiper texto. Descreve a estrutura que uma página web deve ter, sendo esta informação interpretada pelo navegador em uso pelo utilizador.

Caso tenha interesse, pode ler as explicação detalhada para a dissecação dos conceitos de linguagem de marcação e de hipertexto ou pode avançar para os básicos de HTML.

Explicação detalhada

Linguagem de marcação

Em manuscrita, linguagem de marcação consiste num sistema de anotação de um texto para que este seja sintaticamente distinguível, tendo esta ideia surgido das marcações feitas por revisores de manuscritos, normalmente com uma caneta de cor diferente. O objetivo é que o conteúdo destas marcações seja de alguma maneira aplicado para alterar o conteúdo do texto em si, sem estas marcações serem vistas pelo leitor final. Em Informática, linguagem de marcação é um conjunto de códigos aplicados a um texto ou a dados, com o fim de lhe adicionar informações particulares.

Hipertexto

O hipertexto tem as suas origens nós secúlos XVI e XVII, mas é mais simples compreender com os exemplos da internet com que estamos tão familiarizados atualmente.

Em suma, hipertexto é normalmente composto por texto, mas pode consistir em outros elementos visuais, como imagens, que têm uma ligação que nos redireciona para outro conteúdo após clicar.
São mais comummente denominados por hiperligações, ou do inglês, links.

Em suma

Assim, pode-se entender HTML como a linguagem interpretada pelos diferentes navegadores web para estruturar uma página e apresentar o conteúdo a não ser interpretado ao utilizador, incluindo, mas não limitado, a hiperligações.
