Elementos base de uma estrutura html

Independentemente dos conhecimentos que uma pessoa possa ter, qualquer ser humano é falível, e quanto mais acreditamos que dominamos os básicos, mais arriscamos a esquecermo-nos de algum detalhe destes.

Como tal, esta página é um mero lembrete dos elementos HTML imprescindíveis para o devido funcionamento de uma página web. Outros pormenores para uma estrutura mais geral, assim como formatação de elementos e outros é explicado nesta página.

<!DOCTYPE html>

Apesar da semelhança de <!DOCTYPE html> com outros elementos HTML, isto não é um elemento HTMl, mas antes uma informação passada ao navegador acerca de que tipo de ficheiro este é, assim como a sua versão, de modo a que qualquer navegador saiba que tipo de documento deve renderizar e usar os recursos apropriados a esse documento. Como tal, este deve surgir antes de qualquer elemento HTML, sendo normalmente a primeira linha do documento.

À semelhança de qualquer elemento HTML, esta instrução não tem sensibilidade à capitalização usada no elemento, de moco que qualquer uma é devidamente processada pelos navegadores web. Contudo, o padrão mais comummente usado é <!DOCTYPE html>.

Existem várias versões desta diretiva, mas para HTML 5 basta limitarmo-nos a <!DOCTYPE html>. Para versões anteriores de HTML ou diferentes versões de XHTML, pode-se consulta esta página, onde se procura explicar todos os detalhes desta diretiva.

Exemplo de DOCTYPE 

<DOCTYPE html>

<html>

Este deve ser o primeiro elemento HTML propriamente dito, a englobar todos os outros elementos.

O elemento HTML deve ser seguido de um elemento <head>, seguido por um elemento <body>

Dos atributos que podem ser incluídos neste elemento, No mínimo dos mínimos, é altamente aconselhável usar o atributo lang para especificar a linguagem do documento em questão, que ajuda com SEO e com conteúdo localizado. Outros atributos são referidos e explicados aqui.

<html> 

<html lang="pt">

<head>

Inclui a metainformação (informação acerca de informação) da página.

De entre os vários elementos e atributos que podem ser incorporados, o mínimo aconselhável é a metainformação para a codificação de caracteres, responsividade da página e compatibilidade para versões anteriores do internet explorer, caso se queira dar compatibilidade a essas.

Em termos de elementos, devem ser ao menos incluídos <title><link> para incluir ficheiros necessários, assim como <script> para a inclusão de código. Este ponto pode lentificar o carregamento da página, de modo que pode ser preferível colocar no fim do documento.

Podem-se ver os vários atributos e elementos adicionais que podem ser incluídos aqui;

Exemplo de um <head> 

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>
    Exemplo de Head
  </title>
  <link rel="stylesheet" href="estilos/estilos.css">
  <script src="scripts/test.js">
  </script>
</head>

<body>

Não há muito a dizer. É aqui que se insere o conteúdo visto pelo utilizador e todos os outros elementos HTML. Embora tenha alguns atributos específicos que podem ser vistos aqui, não existe nenhum atributo obrigatório.

Exemplo de um <body> 

<body>
  <h1>
    Título
  </h1>
  <p>
    Algum texto
  </p>
</body>

Em suma

Devem-se incluir sempre as tags e os elementos <!DOCTYPE html>, <html>, <head>, <title> e <body> para o tipo de documento ser reconhecido pelo navegador e devidamente estruturado.
